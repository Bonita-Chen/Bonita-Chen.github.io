## Overview

Built and documented a research workflow for tracking AI adoption timing across roughly 900 hospital systems. The project involved entity resolution across multiple data sources and creating reproducible pipelines for longitudinal analysis.

## Key Contributions

- Designed and implemented entity resolution pipelines to match hospital systems across disparate datasets
- Built reproducible Python and Stata workflows for tracking adoption timing patterns
- Created comprehensive documentation enabling other researchers to extend and maintain the analysis
- Processed and cleaned data covering approximately 900 hospital systems nationwide

## Overview

Led a team using R, logistic regression, and visualization to show BMI thresholds vary across racial groups. This project was part of the UMN Public Health Equity Data Challenge.

## Key Contributions

- Led a multidisciplinary team through data analysis, visualization, and presentation
- Applied logistic regression models to demonstrate that BMI thresholds for health outcomes differ by racial group
- Created data visualizations in R that clearly communicated findings to non-technical audiences
- Highlighted equity implications of using uniform BMI cutoffs in clinical settings
